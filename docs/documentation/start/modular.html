Bulma 的元素和组件在大量的 .scss 文件中定义，这些文件您可以 通过使用 @use 关键词单独加载。

{% highlight sass %}{{ use_rule }}{% endhighlight %}

这将正确导入目标文件样式，绝大部分 Bulma 组件依赖默认主题中定义的 base style（基本样式）和 CSS 变量。

这就是也需要导入 sass/base 文件夹和 sass/themes 文件夹的原因：

{% highlight sass %}{{ base_loads }}{% endhighlight %}
{% include docs/elements/anchor.html name="导入列（columns）" %}

Bulma 的布局（Layout）功能不依赖主题的 CSS 变量。所以，可以直接加载而不需要其它文件。

例如：导入 Bulma columns 只需加载文件夹 bulma/sass/grid。

简单的使用 @use 关键词加载文件 columns.scss

{% highlight sass %}{{ import }}{% endhighlight %}

现在您就可以直接使用类 .columns（用于容器）和 .column：

{% highlight html %}{{ columns }}{% endhighlight %}
{% include docs/elements/anchor.html name="导入 Bulma 元素和组件" %}

要加载 Bulma 元素（如 .button）和组件（如 .message），要优先加载基本样式和默认主题。

{% highlight sass %}{{ only_button }}{% endhighlight %}

现在您就可以使用 .button 类和它的修饰符了：

  • .is-active
  • .is-primary, .is-info, .is-success...
  • .is-small, .is-medium, .is-large
  • .is-outlined, .is-inverted, .is-link
  • .is-loading, [disabled]
