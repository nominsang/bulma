--- title: 迁移到 Bulma v1 layout: docs theme: start doc-tab: 起步 doc-subtab: 迁移 breadcrumb: - home - documentation - start - start-migrating --- {% capture markdown %} Bulma v1 是一个用 [**Dart Sass**](https://sass-lang.com/dart-sass/)（该系统是 Sass 的基本扩展）**完全重写**的框架。这可能会影响一些开发细节，但一切已做好准备，确保**平滑**迁移。 {% endcapture %} {% include markdown.html content=markdown %} {% include docs/elements/anchor.html name="相同点" %} {% capture markdown %} **所有 HTML 代码片段不变**。这就是说您不需要更新您的网页内容。**这很重要**因为这意味着如果您在使用 Bulma 的“开箱即用”，您就不需要做任何修改。 您只需将 `bulma@0.9.4/css/bulma.min.css` 改成 `bulma@1.0.0/css/bulma.min.css` 即可。可能看起来略有不同，但正常显示没问题。 {% endcapture %} {% include markdown.html content=markdown %} {% include docs/elements/anchor.html name="不同点" %} {% capture markdown %} * [**Dart Sass**](https://sass-lang.com/dart-sass/) 用于构建 Bulma * 如果您用 `sass` npm 包，您已经在用 Dart Sass * [**CSS 变量**]({{ site.data.links.by_id['features-css-variables'].path }}) 替换了字面变量：`color: var(--bulma-primary);` 不再用 `color: hsl(171deg, 100%, 41%);`，这意味着您可以定制 Bulma，只需用 CSS（而不是 Sass） * 通过设置自己的 Sass 变量值定制的方式不同。查阅[如何用 Sass 定制 Bulma]({{ site.data.links.by_id['customize-with-sass'].path }})。 {% endcapture %} {% include markdown.html content=markdown %} {% include docs/elements/anchor.html name="上新功能（以前没有）" %} {% capture markdown %} * [**主题**]({{ site.data.links.by_id['features-themes'].path }})概念介入：一个主题就是一组相关的 CSS 变量集合，而且是可定制的 * 所以，增加了[**暗色模式**]({{ site.data.links.by_id['features-dark-mode'].path }})主题 * [**调色板**]({{ site.data.links.by_id['features-color-palettes'].path }})针对 7 种主色创建 * [**框架预览**]({{ site.data.links.by_id['features-skeletons'].path }}) 作为一个独立组件出现，但是也会根据其他组件提供变体 * 您可以为您的所有 Bulma 类添加一个**前缀**，所以 `.button` 变成了 `.my-prefix-button` * 一个预制前缀版作为[**备选版本**]({{ site.data.links.by_id["start-alternative-versions"].path }})之一出现 {% endcapture %} {% include markdown.html content=markdown %} {% include docs/elements/anchor.html name="重大变更" %}
