Bulma 中每个元素都是移动设备优先并且针对垂直阅读进行了优化，所以说默认在移动设备使用：

  • columns 会垂直堆叠
  • level 组件会垂直堆叠显示其子元素
  • nav 菜单会隐藏

您可以通过使用 is-mobile 修饰符强制 columns 或 level 使用 horizontal（水平）布局。
